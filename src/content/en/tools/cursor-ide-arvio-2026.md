## What is it?

Cursor is an AI-assisted code editor with chat, Composer, and agent tools wired directly into project context.

## Who is it for?

- AI builders and full-stack developers who want faster refactoring and test writing
- Teams with clear security and data policies for cloud models

## Limitations

- Requires network access to cloud models (unless local models are configured)
- Pricing and data handling must be understood before enterprise use

## Conclusion

A strong **Cursor-first** tool for Faktum AI’s own publishing pipeline; it does not replace code review or testing.
